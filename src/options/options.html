v1.5.9

My Web Shortcuts is an open-source browser extension that lets you create custom keyboard shortcuts in any website to click (or perform other actions) on any element(s).

Note: This extension is currently in its early development phase, you may face a few bugs and a lack of features. If you face any issue, kindly report it to us immediately.

What's New? (v1.5.9) 🔥

Guess what? My Web Shortcuts just got cozier with Globy & Keyby at our winter campfire! Big things are brewing. Wanna help fuel the fire? Consider sponsoring us—it keeps the good vibes flowing. Stay warm and stay tuned for more fun! 😊
  • 🟢 You can finally Import and Export your Web Shortcuts!! Go to Global settings and click on the Import/Export buttons to do so.
  • 🟡 I have a new shortcut to create a new Web Shortcut, and that is Ctrl/Command + Shift + Space. It's super quick and handy, open a website and try it out!
  • 🔴 There was a perfomance bug in me slowing down your browser. But my brave developers fought a huge battle and killed it in the end! I am so proud of them.
  • 🟢 There's now a "Quick Help" section in the Popup to get some, umm, quick help lol. A few other things have been implemented to help the new users.
  • 🟢 While using Web Shortcuts, there will be visual indicators (red circle) appearing over the elements.
  • 🟡 Now there are more shortcuts shown in the Popup, up to 4 at a time. It was only up to 2 at a time previously.
  • 🟡 Pausing Element selection was a bit irritating as it used to immediately pause/unpause on the Ctrl/Command key press. But now you need to Double press the Ctrl/Command key to pause/unpause Element Selection.
  • 🔴 Mac Users can now use the Pause Element Selection feature too. There was a bug that didn't allow them to use it till now.
  • 🟢 You can jump to Shortcuts settings, Global settings, and About using 1, 2, and 3 keys respectively.
  • 🔴 You can now use Tab key with Enter/Space keys to navigate the websites list. There was a bug that didn't allow one to open a website using those keys.
  • 🔴🟡 A few other bug fixes & improvements, and...
  • 🔵 New Winter look! It's cool isn't it? or is it warm? Serious questions.
